# Team Roles & Technical Breakdown

This document outlines the core roles, responsibilities, and deliverables for each team member in the project.

***

## Person 1 — Backend & AI Lead

**Role:** Data + Credit Intelligence

### Responsibilities
- Integrate **Pay Hero API**  
- Build **sales ingestion pipeline**  
- Design and optimize **Trust Score algorithm**  
- Implement **risk engine logic**  
- Define **credit approval and validation rules**  
- Develop **repayment calculation engine**  
- Expose analytics through **Merchant dashboard API**

### Tech Stack
- **Node.js / NestJS / python** – backend and API framework  
- **PostgreSQL** – relational data layer  
- **Python** – AI-based scoring and predictive models  
- **Redis** – caching and real-time computation

### Deliverables
- **Trust Score API**  
- **Credit Limit Calculator**  
- **Repayment Percentage Engine**  
- **Merchant Risk Monitoring System**

***

- Build **automated routing and contract settlement logic**

### Tech Stack
- **Stellar SDK**  
- **Rust (Soroban)** – smart contract language  
- **Horizon API** – blockchain interaction layer  
- **Wallet Key Management System**

### Deliverables
- **Smart Contract Template**  
- **Automated Payout Routing**  
- **Blockchain Transaction Handler**  
- **Escrow and Settlement Logic**
